## DEVELOPE AND CONTRIBUTE 👩‍💻👨‍💻

### SETUP

	git clone git@github.com:deesouza/React-Cartola-FC.git

Or **Fork** Project.

### INSTALL DEPENDENCIES

	cd React-Cartola-FC && yarn

### RUN

	yarn start

### BUILD

	yarn build
